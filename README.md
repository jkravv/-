**Αυτόνομο Ρομποτικό Όχημα Αγρότης (Α.Ρ.Ο.Α.)**

Ένα αυτόματο ρομποτικό όχημα για την χαρτογράφηση του εδάφους στο χωράφι και για βοηθητικές εργασίες.


**Σύνδεση με το πρόγραμμα σπουδών στο Δημοτικό.**

Σύμφωνα με τις οδηγίες διδασκαλίας και διδακτέα ύλη Δημοτικού Σχολείου για το διδακτικό μαθησιακό αντικείμενο  «Τεχνολογίες Πληροφορίας και Επικοινωνιών» (Αρ. Πρωτοκόλλου: Φ20/1835/222136/Δ1) για την Ε' και Στ'  Τάξη προβλέπεται η ενότητα «Υλοποιώ σχέδια εργασίας/έρευνας (project) με τις ΤΠΕ» με ενδεικτικό διδακτικό χρόνο 8 ώρες. Μέσα στις προτεινόμενες δραστηριότητες υπάρχει καταγεγραμμένη και η Εκπαιδευτική Ρομποτική στην οποία ανήκει το παρόν project.
